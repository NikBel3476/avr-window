Чирков Борис Владимирович

Цель курсовой работы

Спроектировать устройство для автоматического управления положением подвижной створки окна

Основные задачи

  1. Спроектировать общую схему устройства
  2. Спроектировать механизм
  3. Написать программу для управления
  4. Собрать всё в готовое решение

Общая схема устройства

Управляющий механизм

Свойства

  • Простым
  • Легко устанавливаться на окно
  • Надёжным(по возможности)

Первый вариант

Механизм на основе катушки с леской

Недостатки первого варианта

  • Не все окна открываются под собственным весом
  • Сложность регистрации момента открытия окна

Второй вариант

Механизм на основе зубчатой рейки и шестерни

Преимущества перед первым вариантом

  • Можно как открывать, так и закрывать окно
  • С помощью герконов можно регистрировать факт того, что окно закрыто, либо открыто

Микроконтроллер

Atmega128rfa1

Программа для управления микроконтроллером

Задачи:

  • Управлять механизмом открывания/закрывания
  • Отсчитывать необходимое для проветривания время
  • Обрабатывать сигналы с устройства передачи данных
