Отчет по курсовой работе

Цель курсовой работы

Спроектировать устройство для автоматического проветривания помещения путем открытия и закрытия окон и дверей

Задачи курсовой работы

  1. Спроектировать общую схему устройства
  2. Спроектировать механизм
  3. Написать программу для управления
  4. Собрать всё в готовое решение
