Отчёт по курсовой работе

Цель курсовой работы

Спроектировать устройство для автоматического проветривания помещения путем открытия и закрытия окон

Задачи курсовой работы

  1. Спроектировать общую схему устройства
  2. Спроектировать механизм
  3. Написать программу для управления
  4. Собрать всё в готовое решение

Первый вариант

Механизм на основе катушки с леской

Недостатки первого варианта

  • Как открыть окно?
  • Как регистрировать момент полного открытия окна?

Второй вариант

Механизм на основе зубчатой рейки и шестерни

Преимущества перед первым вариантом

  • Можно легко как открывать, так и закрывать окно
  • С помощью герконов можно регистрировать факт того, что окно закрыто, либо открыто

Текущая логика работы программы

  1. Нажатие на кнопку -> окно открывается
  2. Срабатывает геркон в конце рейки -> механизм останавливается
  3. Нажатие на кнопку -> окно закрывается
  4. Срабатывает геркон в начале рейки -> механизм останавливается
